初识JAVA

实不相瞒，最早接触这个词，是在小时候，拿爸爸的诺基亚打开小游戏，开屏即是一杯咖啡。（附图）

那时候在想，这个咖啡广告居然在游戏开屏，真是不可思议。
简而言之，对编程者来说，JAVA是语言，是我们和应用程序交流的工具。我们使用java进行编程，类似于写作，用中文进行写作，写下的文字便像拥有了生命，鲜活地存在在世界上。
对于计算机，它需要理解我们所说的话，然后做出相应的反应。咿呀学语，编程界总是习惯于从hello world开始第一篇。
