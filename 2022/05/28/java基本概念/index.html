java基本概念

0001 JRE JDK JVM

  1. JDK java development kit， 包含JRE和开发所用，包括编译器，可以创建和编译。
  2. JRE java runtime environment, 包含jvm、java类库、java命令等，有就可以运行java程序。（只运行不创建）
  3. JVM java虚拟机，运行字节码的。字节码在jvm上运行，不同操作系统上的jvm实现不一样，但同一字节码不同jvm运行结果一致。

0002 面向对象是什么？和面向过程的区别。

面向对象是一种解决问题的思想，编程的思想。从原先只注重流程化的工作中抽象出来一个个对象，界定好对象的成员和职责。对象提供操作方法，我们只需要关注对象和对象之间关系就可以写代码。而对于面向过程，我们只注意具体流程设计，不太考率复用性和组件性。

0003 java语言特点

  1. 面向对象（封装、继承、多态）
  2. 一次编译、到处运行。平台无关
  3. 先编译成class, 再用jvm解释执行

0004 java和C++区别

  1. c++有指针
  2. java有自动内存回收机制gc，c++要手动
  3. c++可以多继承，java通过借口来实现多继承。

0005 java基本类型

  1. 占字节大小
    ：1248 248
    byte short int long char float double boolean
    不加static修饰的成员变量存放在堆中，局部变量在栈
  2. 装箱与拆箱 Integer.valueOf() Integer.intValue()
    包装类会有缓存范围，范围内则不会创建新对象，范围外则会新建对象。范围外值一样也不会equals.

0006 new一个对象发生的事情

  1. Stu a = new Stu();
    在堆上创建了一个Stu类的对象实例，在栈里创一个引用a,a指向对象实例。

0007 构造方法

  1. 构造方法没有返回值默认无参，可以重载（同名不同参）不能重写。

0008 面向对象的三大特征

  1. 封装、继承、多态
  2. 封装 对外展示是类的方法，内部属性隐藏，不允许直接修改。
  3. 继承 子类继承父类，所有的都会继承，但父类私有的不能动。子类可以自行增加或者重写。
  4. 多态 继承/实现+重写+父类引用指向子类对象
    eg : A a = new B(); B extend A
    eg : 实现接口 B implement A;
    对多态的理解：一处代码，却可以赋予多种实现，程序运行时才确定到底执行睡。

0009 接口和抽象类有什么区别？

  1. 都是用来声明某一类规则的，所以是虚的，不能实例化、可以有抽象方法，也可以有默认实现。
  2. 接口只要是行为约束，抽象类更多是面向对象的所属关系。
  3. 一个类可以实现多个接口，但是只能继承一个抽象类。
  4. 接口中的变量是固定的,public static final, 必须初始化并且不可改变，抽象类随意。（此处疑问？为什么要引入默认方法？–临时解答：为方便在原有接口上增加功能。）

0010 深拷贝、浅拷贝、引用拷贝

  1. 该问题的研究对象： 引用（变量）T——> A对象（B对象为内部对象）
  2. 深拷贝: TAB三者独立
  3. 浅拷贝: T和A都拷贝，比深要浅一层；
  4. 引用拷贝: 只拷贝引用T，AB不动

0011 hashcode equals

  1. equals > hashcode, 重写equals必须重写hashcode.

0012 String

  1. 字符类 是一个类而不是基本数据类型，所以要用对象的眼光去看待。
  2. 字符串，是具体String对象的值。常量池存储。
  3. String 不可变，是因为final修饰不能被继承+内部字符数组不能被继承也不提供对外修改方法的共同作用。所以线程安全。
  4. StringBuilder 可变 线程不安全

0013 异常

  1. error 程序无法处理 OOM/ exception
  2. exception分为 运行时异常和可检查异常
  3. 运行时异常是指要到运行的时候才能发现的，可检查异常
